# eTransit---Smart-Bus-Pass-App
eTransit – Smart Bus Pass App is a digital solution for managing bus passes through a mobile application. It allows users to apply, renew, and store their bus passes online. The app reduces paperwork, saves time, and makes public transport more convenient by providing a secure and easy way to access and verify bus passes digitally.

link - http://127.0.0.1:5500/index.html

Key Features
1. Digital Bus Pass -
Users can create and store their bus pass digitally in the app, reducing the need for physical passes.
2. Online Pass Renewal -
Passes can be renewed online without visiting bus stations.
3. User Registration & Login -
Secure login system for users to manage their personal travel passes.
4. Pass Validity Tracking - 
Users can easily check the validity and expiry date of their bus pass.
